# Breakout 3D Game
Breakout 3D Game made with OpenGL as course project for the course Computer Graphic 2021 at Technical University of Sofia 

## :hammer: Used technologies
* C/C++
* OpenGL
* stb_image
* glad
* GLM
* Assimp
